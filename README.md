# Finca El Carmen

Proyecto React + Vite + Firebase + Vercel.

## Instalación

```bash
npm install
cp .env.example .env
npm run dev
```

## Build

```bash
npm run build
```

## Importante

- No subir `.env` a GitHub.
- Configurar variables `VITE_FIREBASE_*` en Vercel.
- Publicar `firestore.rules` y `storage.rules` en Firebase.
